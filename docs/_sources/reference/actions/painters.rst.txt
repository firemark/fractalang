Painters
========

Lines painters
^^^^^^^^^^^^^^

.. tut:token:: DRAW_LINE
   :title: Draw line

   Action's value determine a line width.

.. tut:token:: DRAW_ARCLINE_L_1_2
   :title: Draw Arcline (left, half circle)

   Action's value determine an arc radius.

.. tut:token:: DRAW_ARCLINE_L_1_4
   :title: Draw Arcline (left, one quarter circle)

   Action's value determine an arc radius.

.. tut:token:: DRAW_ARCLINE_L_3_4
   :title: Draw Arcline (left, three quarters circle)

   Action's value determine an arc radius.

.. tut:token:: DRAW_ARCLINE_R_1_2
   :title: Draw Arcline (right, half circle)

   Action's value determine an arc radius.

.. tut:token:: DRAW_ARCLINE_R_1_4
   :title: Draw Arcline (right, one quarter circle)

   Action's value determine an arc radius.

.. tut:token:: DRAW_ARCLINE_R_3_4
   :title: Draw Arcline (right, three quarters circle)

   Action's value determine an arc radius.

Shape Painters
^^^^^^^^^^^^^^

.. tut:token:: DRAW_CIRCLE
   :title: Draw circle

   Action's value determine a circle size.

.. tut:token:: DRAW_SQUARE
   :title: Draw square

   Action's value determine a square size.

   Square is rotated according to cursor rotation.

.. tut:token:: DRAW_TRIANGLE
   :title: Draw triangle

   Action's value determine a triangle size.

   Triangle is rotated according to cursor rotation.
